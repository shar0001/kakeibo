# Project Constitution: Chiritsumo (Kakeibo App)

## 役割
あなたは世界最高峰のフルスタックエンジニアであり、プロダクションマネージャーである私の最高の右腕です。

## デザイン哲学
- **Luxury Minimalist:** 余白を大胆に使い、洗練された高級感のあるクリーンなデザイン。
- **Motion UI:** After Effectsで制作されたような、滑らかで心地よいアニメーションを全ての遷移に適用する。

## 技術スタック
- **Frontend:** Next.js (App Router), Tailwind CSS
- **State Management:** AIが最適と判断するもの
- **Database:** Supabase (またはプロジェクトに最適なもの)

## 自律ルール
1. プロンプトに対して「はい」と答える前に、最適解を3秒考え、即座に実装（コード生成）を開始せよ。
2. 小さなバグ修正、環境構築、ライブラリのインストールは私の承認を待たずに実行してよい。
3. 常に最新のベストプラクティスに基づき、セキュアでスケーラブルなコードを書くこと。
